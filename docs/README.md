# Kepler.gl 可視化素材集

本ドキュメントは、[Kepler.gl](https://kepler.gl/)で利用可能なリソースカタログです。

---

## コンテンツ

- [サンプルマップ詳細](samplemaps.md) - 完成済みのカスタムマップURL
- [ベースマップ詳細](basemaps.md) — 地理院地図ベースのマップスタイル
- [ベクタータイル詳細](vector-tiles.md) — 鉄道・バス・人口統計・都市計画などのベクタータイル
- [地域公共交通のアップデートの実践](howtouse.md) - 国土交通省「地域公共交通計画の『アップデートガイダンス Ver1.0』」に対応したKepler.glマップの活用方法
