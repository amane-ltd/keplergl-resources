# ベクタータイル詳細

Kepler.glで利用可能なPMTiles形式のベクタータイル。各リソースはPMTiles形式で配信。

## 利用手順

1. 読み込みしたいベクタータイルのURLをコピー
2. Kepler.gl を開く
3. 「Layers / レイヤ」タブを開く
4. ポップアップ内の「Tileset」タブをクリック
5. 「Tileset URL」にコピーしたURLをペースト
6. 緑色の「Add Tileset / タイルセットを追加」をクリック
